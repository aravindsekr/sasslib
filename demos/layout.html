How to use Viewports

Some popular viewport sizes are:

  • 360×640
  • 1366×768
  • 1920×1080
  • 375×667
  • 1440×900
  • 1280×800

We have breakpoints at 768px, 992px and 1200px

The styles are Mobile first and to override in desktop, use these mixins.

@include large-mobile { ... } @include tablet { ... } @include desktop { ... }
